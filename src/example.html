Literature Management: Web Interface

The literature is managed through the subversion repository located at https://optimierung.mathematik.uni-kl.de/repos/literature. Checkout the repository in order to add papers and use the BibTeX database in your own work. An up-to-date, read-only checkout is automatically generated under "MAB" ("I:") in the folder "Literature". There you also have file-based access to all PDF documents.

For more information on how to use the literature repository, see this document.

The .bib files can also be downloaded here: agopt.bib (main database), abbr.bib (abbreviated journal names), and full.bib (full journal names).
